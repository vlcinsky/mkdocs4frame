<u>Technical note: </u><br/>An instance of the abstract <font color="#800000">OperatorAction</font> class needs a specialisation, which – for technical reasons – cannot be fulfilled by a Level B extended class such as the <font color="#800000">SignSetting </font>class. For this reason, when using this Level B extension, the <font color="#800000">GeneralNetworkManagement</font> of type <font color="#4169e1">other</font> should be used (for the future, it is intended to transform the Level B extension into inheritance).<br/><br/><br/>
